# A Lipschitz spaces view of infinitely wide shallow neural networks

* F. Bartolucci, M. Carioni, J. Iglesias, Y. Korolev, E. Naldi, S. Vigogna. A Lipschitz spaces view of infinitely wide shallow neural networks (2024). ArXiv preprint.
